# Calculator
A simple calculator app built with HTML, CSS, and JavaScript that allows users to perform basic mathematical operations such as adding, subtracting, dividing, and multiplying.  
## Features
* Basic arithmetic operations (addition, subtraction, multiplication, division)  
* Keyboard support for buttons  
  * `0-9` to enter numbers  
  * `.` to add a decimal point  
  * `+`, `-`, `*`, `/` to add, subtract, multiply, divide  
  * `enter` to display the result  
  * `backspace` to clear entry  
  * `esc` to clear the display
